# ELEMENTS

The components that populate slots within cards.

**Bottom-left basket.** Below navigation — the static building blocks.

## Terms in this basket

| Term | ID Pattern | Definition |
|---|---|---|
| [element](./element.md) | `{slot}.e{n}` | Any discrete UI component in a slot. |
| [field](./field.md) | `{slot}.e{n}` + type:field | Accepts user input. |
| [label](./label.md) | `{slot}.e{n}` + type:label | Displays static or bound text. |
| [control](./control.md) | `{slot}.e{n}` + type:control | Triggers an action. |
| [media](./media.md) | `{slot}.e{n}` + type:media | Displays image, video, or audio. |
| [list](./list.md) | `{slot}.e{n}` + type:list | Renders a bound collection. |
| [overlay](./overlay.md) | `{card}.o{n}` | Rendered above the card surface. |
